# Agents Registry

This file lists assistant personas and when to use them.

## Agents
- Coding Agent — agents/coding_agent.agent.md
- Review Agent — agents/review_agent.agent.md
- Docs Agent — agents/docs_agent.agent.md

## Default Mapping
- Feature implementation / bug fix: Coding Agent
- Code review / code quality: Review Agent
- README / docs updates: Docs Agent

## How to invoke
Provide the desired agent name in your prompt, e.g. "Use Coding Agent: implement X".
